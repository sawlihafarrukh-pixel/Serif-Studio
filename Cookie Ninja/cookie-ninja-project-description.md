# Cookie Ninja — Portfolio / Project Description

For Upwork, Contra, and portfolio case studies. Frame: **Social Media & Brand Photography**.

## Main description

> Cookie Ninja is a home bakery brand for which I managed social media and led product photography. My role involved creating a consistent visual identity across all content, shooting and editing product photos to highlight the brand's handmade quality, and developing a content strategy to grow engagement. All photography was styled, shot, and edited by me using Lightroom.
